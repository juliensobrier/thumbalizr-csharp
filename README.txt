Thumbalizr C# library
================================


Thumbalizr (https://thumbalizr.com/) is a web service to easily make screenshots of web pages. Thumbalizr has full Flash, JavaScript, CSS, & HTML5 support.


The API is detailed at https://api.thumbalizr.com/. 


The source code can be found on github at https://github.com/juliensobrier/thumbalizr-csharp
The documentation can be generated from the source code with Doxygen, or viewed at http://juliensobrier.github.com/thumbalizr-csharp/
